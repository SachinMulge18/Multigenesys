# MultiGenesys Task

A simple employee directory built with React, MUI, RTK‑Query and Vite.  
Includes CRUD operations against a mock API and a responsive form/table UI.

## 🛠 Project setup

git clone https://github.com/SachinMulge18/Multigenesys.git
cd MultiGenesysTask
npm install
npm run dev
npm run build
npm run test
